What is the CCC?

The Canadian Computing Competition (CCC) is a programming contest for students in Canada. It tests problem-solving skills and programming knowledge using Python, Java, and C++.

Preparing for the CCC

The Canadian Computing Competition (CCC) helps students improve programming and problem-solving skills through practice.

Why Prepare?

Preparing for the CCC helps improve logical thinking, coding skills, and confidence. It’s a great way to get ready for future computer science courses.
