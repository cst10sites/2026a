Common HTML Tags Outside <body>

<body> Tags Definition Example(s)
<!DOCTYPE html> Defines the document type and version of HTML being used.

No examples
<html lang="en"> This page is set to "en" for English. Helps browsers and translation tools know the default language. This page is set to “en” for English. This tag helps websites such as Google Translate know the default language for this page is English.
<meta charset="UTF-8"> Specifies character encoding for the HTML document No examples
<meta http-equiv="refresh" content="0; url=URL"> Redirects the page after a set time or immediately. No examples
<title> </title> Section that describes the title of the page shown on the browser tab.
<style> </style> Used to describe internal CSS rules within the page. Show <style>...</style> that makes this cell background black using id or group.
<link rel="stylesheet" href="styles.css"> Tag that points to an external CSS file The homepage and this page point to a common style.css file that controls the look of these pages.
<link rel="icon" href="favicon.png> Adds a small icon (favicon) to the browser tab
