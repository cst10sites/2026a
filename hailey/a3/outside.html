<!DOCTYPE html>

It gives the computer information on what type of document it is.

No examples
<html lang="en">

This tags tells the website that the default language is English.

This page is set to "en" for English. This tags helps websites such as Google Translate know the default language for this page is English.
<meta charset="UTF-8">

The tag is used to specify the character encoding for the document, this ensures that text is displayed correctly, especially when using languages that do not use the latin alphabet.

No examples
<meta http-equiv="refresh" content="0; url=$URL">

Refreshes your page after a given time interval.

No examples
<title>...</title>

Creates a title for your page.

<style>...</style>

Section that describes the internal CSS of the page.

<link>
