Why I took this course/What I learned so far

Since Elementary school, coding has always been something that caught my attention. It's something I have the desire to learn, yet feels to far to take the first step on learning how to code. It looked ridiculously hard to code. In my young mind, the phonomena doubled. The point is I always wanted to learn how to code. As I know the world is in an immense shift to a digital age, and furthermore the appearance of A.I technology, I knew knowing how to code will be a valuable asset. I knew this, but I thought it wasn't the right time yet. You know, I thought I had more important things to do, like math, science or English. I did learn how to use scratch, but it didn't really satisfy my need to learn how to code. Time passed until I decided to operate my own business. And upon trying it, I realized this is the time I need to learn how to code. Now or never. Primarily, I took CST 10 to build my own website. However, deep down inside, I knew I was procrastinating the urge to learn how to code, and that I always wanted to learn how to code. I just thought it wasn't the time yet. So far, even though it has officially been only a week since I learned how to code, where I understand how to speak basic vocabularies in html, or how coding starts, how it works, and how to end it, I have never been this delighted to learn and see my codes actually work.
