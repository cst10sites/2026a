HTML tags Definitions Examples
<!DOCTYPE html> This is the very first line of the code, for every html codes. This line of tag declares what system of code the computer or the webpage will use. By using <!DOCTYPE html>, This computer understands that the code language I am using to code is html, or one of the versions of html.
<html lang="language"> This tag defines what language the code is using This code enables the computer to understand that the language used in this code is based on English.
<meta charset="UTF-8"> This tag takes on the UTF-8 system where it enables the computer to read extraordinary characters. The system UTF-8 allows the computer to read unordinary characters such as 안녕하세요, another language. This is not legible in different system, such as the Window 1252.
<title></title> This tag is responsible for the title of the webpage The title of the website defines what will be shown in the top nav bar of the website. If you look up, you can see Assignment 3, which is the title of the code.
<style></style> This tag is responsible for the internal CSS, used inside index.html This has been modifed by internal CSS
<link rel="stylesheet" href="styles.css"> This tag reaches external CSS outside of index.html, where external CSS ususally has more features than internal CSS. Most of the time, external CSS allows more organized, and various codes of CSS to be used. This code is currently using external CSS, which I wrote down the code for CSS on another code page, which is styles.css.
<link rel="icon" type="image/x-icon" href="favicon.ico"> This tag is used for favicon, which is the small icon you see next to the title of the webpage. The one you see next to the title is the favicon. I used this code to generate that image to be favicon.
<head></head> This tag creates a section called "head" like "body", where the main system of how the code is going to be presented is. Just like the brain of a human being. All these codes presented in this table belongs in <head></head>.
