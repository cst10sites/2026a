Why I took this course?

I was enrolled in Computer Studies 10 based on a recommendation from my school counsellor, and my parents were very supportive, believing it would be a valuable and useful skill for me to learn. While I didn't choose it entirely on my own initially, I was quickly interested because I’ve always wanted to understand how they are made. I also agreed that learning to code is an important skill for almost any future academic path or career, and I saw this course as a great opportunity to get a head start and see if it's something I'd want to pursue further. So far, our main project has been building a game called "Catch the Clown" in GameMaker, which has taught me the basics of how games work. I’ve learned how to use the GameMaker interface, create objects, and assign them sprites. More importantly, I’ve learned core programming concepts by actually typing the code myself, such as using variables to store information like the score and speed, and using events like the Step Event to make the clown move randomly and the Mouse Pressed Event to make the score increase and play a sound when he’s clicked.

This hands-on project showed me that programming isn’t just about typing complex code; it’s a creative process of logical thinking and problem-solving. Figuring out why the clown wasn’t moving correctly or getting the score to update properly felt like solving a puzzle. I'm glad I was placed in this course because it has demystified how software is built and has given me a practical and satisfying foundation in a key skill for the future.
