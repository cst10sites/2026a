Tag Definition Examples
<!DOCTYPE html> It must be the very first line in an HTML document, before the <html> tag. It is not an HTML tag; it’s a declaration. It ensures that the browser renders the page in standards mode No examples*
<lang="language"> This line sets the language of the page. This page is set to “en” for English. This tag helps websites such as Google Translate know the default language for this page is English.
<title></title> This tag makes the tile at the top of the webpage.
<meta charset="utf-8"> This tag makes some funky charecters legible. No examples*
<meta http-equiv="refresh" content="0; url=$URL"> This is a meta refresh tag, used to automatically redirect the browser to another web page after a specified number of seconds. No examples*
<link rel="stylesheet" href="styles.css"> Tag that points to an external CSS file The homepage and this page point to a common style.css file that controls the look of these pages.
<ink rel="icon" type="image/x-icon"href="favicon.ico"> Tag that adds a favicon
<style></style> Section that describes the internal CSS of the page This cell is black
