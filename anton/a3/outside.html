Common HTML Tags outside <body>

Tags Definitions Example(s) if applicable
<!DOCTYPE html> This indicates that the type of the file shown is HTML file, which allows use of HTML No examples available.
<html lang="en"> This sets the language of the page to English, so that web translaters like Google know the default language for this page is English. This sentence is English.
<meta charset="UTF-8"> This tag ensures that the browser correctly interprets and displays the characters on the webpage. No examples available.
<meta http-equiv="refresh" content="0"; url=$URL"> Used to instruct a web browser to automatically refresh the current web page or frame after a specified time interval. No examples available.
<title>...</title> Defines the title of a web document, which is displayed in the browser's title bar or on the page's tab.
<style>...</style> This is the section that descrides the internal CSS of the page. This is styled by internal CSS.
<link> This links the seperate CSS file to the HTML file in order to apply the external CSS style to the webpage. <link rel="stylesheet" href="styles.css". This is the code that links. Most styles in this page are styled by the external CSS file that is linked to this HTML file.>
<link rel="icon" type="image/x-icon" href="icons8-porridge.gif"> This tag adds a favicon.
