inside body

Tag Definition Example(s) if applicable
<!DOCTYPE html> This tag tells the browser that the document is written in HTML5. It ensures the page is rendered correctly by putting the browser in standards mode, and it should always appear at the very top of the HTML file. no examples*
<html lang="en"> This page is set to “en” for English. This tag helps websites such as Google Translate know the default language for this page is English. This page is set to “en” for English. This tag helps websites like Google Translate.
<meta charset="UTF-8"> This tag tells the browser to use the UTF-8 character encoding, which supports almost all characters and symbols from every language.
  • Accented letters are not possible without <meta charset="UTF-8">: é, ñ, ü
<title></title> This tag sets the title of the webpage, which appears in the browser tab and is used by search engines as the page’s name. It helps users know what the page is about.
<meta http-equiv="refresh" content="0; url=$URL"> This tag is used to automatically redirect a web page to a different URL after a specified time. The content attribute defines the delay (in seconds) and the destination URL. No examples*
<link rel="stylesheet" href="style.css"> Section that describes the internal CSS of the page. this black background is made from using <style>...</style>
<ink rel="icon" type="image/x-icon"href="favicon.ico"> Tag that points to an external CSS file. The homepage and this page point to a common style.css file that controls the look of these pages.
<style>...</style> Tag that adds a favicon.
